# 💳 Credit Card Fraud Detection using Machine Learning

An AI-powered Credit Card Fraud Detection System built using Python and Machine Learning techniques.  
This project detects fraudulent credit card transactions using classification algorithms and anomaly detection concepts.

---

# 🚀 Features

- Detect fraudulent transactions
- Handles imbalanced datasets
- Machine Learning classification models
- Beginner-friendly implementation
- Fast and lightweight dataset
- Easy to run locally or in Google Colab

- Machine Learning

# 🤖 Machine Learning Model Used

## Logistic Regression

A supervised Machine Learning algorithm used for binary classification problems.

In this project:
- 0 → Normal Transaction
- 1 → Fraudulent Transaction

---

# 📊 Important Concepts

## Imbalanced Dataset

Fraud datasets usually contain:
- More normal transactions
- Very few fraud transactions

---

## Fraud Detection

The model learns patterns from transaction data and predicts whether a transaction is fraud or not.
